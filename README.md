# 🏥 Sistem Pendaftaran Online Pasien - Ponkesdes Landungsari

Sistem pendaftaran pasien berbasis web yang dirancang untuk digitalisasi layanan di **Ponkesdes Landungsari**. Proyek ini mengutamakan kecepatan akses (*Anti-Delay*) dan antarmuka yang modern.

## ✨ Fitur Unggulan
* **Integrasi Real-time**: Data yang diinput otomatis masuk ke Google Sheets via Google Apps Script.
* **Validasi Otomatis**: Memastikan NIK berjumlah 16 digit dan nomor WhatsApp menggunakan format yang benar.
* **UI/UX Modern**: Menggunakan framework Tailwind CSS dengan desain kartu layanan yang intuitif.
* **Animasi Interaktif**: Menggunakan SweetAlert2 untuk memberikan feedback pengiriman data yang menarik.

## 🛠️ Teknologi yang Digunakan
* **Frontend**: HTML5, Tailwind CSS, JavaScript (ES6+).
* **Backend**: Google Apps Script.
* **Database**: Google Sheets.
* **Hosting**: Vercel.
